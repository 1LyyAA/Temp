# Пункт 9: Обучение с учителем (Supervised Learning)

## Основные идеи:

1. **Компоненты задачи:**
   - Обучающая выборка L = {xᵢ, yᵢ}ⁿᵢ₌₁
   - Модель f(x) для предсказаний
   - Функция потерь Q(x, y, f)

2. **Типы задач:**
   - Регрессия: y ∈ ℝ (непрерывные значения)
   - Бинарная классификация: y ∈ {+1, -1}

3. **Цель:**
   - Минимизировать функцию потерь
   - Найти оптимальные параметры модели
   - Обобщить на новые данные

4. **Общая схема:**
   - Данные → Обучение → Модель → Предсказания
