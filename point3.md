# Пункт 3: Теорема Гаусса-Маркова

## Основные идеи:

1. **Условия теоремы:**
   - Линейность модели
   - Экзогенность: E[εᵢ | X] = 0
   - Гомоскедастичность: Var(εᵢ | X) = σ²
   - Некоррелированность ошибок
   - Полный ранг матрицы X

2. **Вывод теоремы:**
   - При выполнении всех условий МНК-оценка является BLUE
   - BLUE = Best Linear Unbiased Estimator
   - Наилучшая линейная несмещенная оценка

3. **Значение теоремы:**
   - Теоретическое обоснование МНК
   - Гарантирует оптимальность оценки при выполнении условий
   - Объясняет, почему МНК так широко используется
