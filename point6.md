# Пункт 6: Метрики качества регрессии

## Основные идеи:

1. **MSE (Mean Squared Error):**
   - Среднеквадратичная ошибка
   - Чувствительна к выбросам
   - Штрафует большие ошибки сильнее

2. **MAE (Mean Absolute Error):**
   - Средняя абсолютная ошибка
   - Более устойчива к выбросам
   - Интерпретируется как средняя ошибка

3. **R² (Коэффициент детерминации):**
   - Показывает долю объясненной дисперсии
   - Значение от 0 до 1 (или отрицательное для плохих моделей)
   - R² = 1 означает идеальное предсказание

4. **MAPE (Mean Absolute Percentage Error):**
   - Средняя абсолютная процентная ошибка
   - Показывает ошибку в процентах
   - Удобна для интерпретации

5. **RMSE (Root Mean Squared Error):**
   - Корень из MSE
   - Имеет те же единицы измерения, что и целевая переменная
