# Пункт 12: Метод главных компонент (PCA)

## Основные идеи:

1. **Назначение PCA:**
   - Снижение размерности данных
   - Борьба с мультиколлинеарностью
   - Визуализация многомерных данных
   - Уменьшение вычислительной сложности

2. **Принцип работы:**
   - Находит направления максимальной дисперсии
   - Первая компонента объясняет наибольшую дисперсию
   - Последующие компоненты ортогональны друг другу
   - Каждая следующая объясняет оставшуюся дисперсию

3. **Свойства главных компонент:**
   - Ортогональность (некоррелированность)
   - Упорядочены по важности
   - Линейные комбинации исходных признаков
   - Сохраняют максимум информации

4. **Применение:**
   - Выбрать первые k компонент
   - Использовать их вместо исходных признаков
   - Решает проблему коллинеарности
   - Ускоряет обучение моделей
