# Пункт 5: Регуляризация (Ridge и Lasso)

## Основные идеи:

1. **Ridge-регрессия (L2-регуляризация):**
   - Добавляет штраф ||ω||²₂ к функции потерь
   - ω̂ᵣᵢdgₑ = (XᵀX + λI)⁻¹Xᵀy
   - Имеет аналитическое решение
   - Ограничивает веса, стабилизирует решение
   - Дифференцируема

2. **Lasso-регрессия (L1-регуляризация):**
   - Добавляет штраф ||ω||₁ к функции потерь
   - Нет аналитического решения
   - Использует численную оптимизацию
   - Выполняет отбор признаков (обнуляет коэффициенты)
   - Недифференцируема (но это не проблема)

3. **Гиперпараметр λ:**
   - Контролирует силу регуляризации
   - Больше λ → сильнее штраф → меньше коэффициенты
   - Подбирается через кросс-валидацию
