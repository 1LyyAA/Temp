# Пункт 4: Проблемы МНК и их решения

## Основные идеи:

1. **Проблемы МНК:**
   - Мультиколлинеарность (высокая корреляция между признаками)
   - Вычислительная сложность O(mn² + n³)
   - Отсутствие защиты от переобучения
   - Единственная функция потерь (MSE)

2. **Решения мультиколлинеарности:**
   - Исключение избыточных признаков
   - Регуляризация (Ridge, Lasso)
   - Метод главных компонент (PCA)

3. **Когда возникает:**
   - Большое количество коррелирующих признаков
   - Малое количество наблюдений относительно признаков
   - Линейно зависимые признаки
