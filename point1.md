# Пункт 1: Матричное представление линейной регрессии

## Основные идеи:

1. **Модель для всех объектов (векторная форма):**
   - y = Xω + ε
   - Матрица признаков X размера n×(p+1)
   - Вектор коэффициентов ω
   - Вектор ошибок ε

2. **Модель для одного объекта:**
   - yᵢ = xᵢᵀω + εᵢ
   - Вектор признаков xᵢ включает 1 для свободного члена

3. **Ключевые компоненты:**
   - n — число объектов в датасете
   - p — число признаков
   - ω₀ — свободный член (intercept)
   - ω₁, ω₂, ..., ωₚ — коэффициенты для признаков
