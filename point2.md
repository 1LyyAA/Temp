# Пункт 2: Метод наименьших квадратов (МНК)

## Основные идеи:

1. **Функция потерь (MSE):**
   - Минимизируем квадраты разностей между реальными и предсказанными значениями
   - J(ω) = ||y - Xω||²₂

2. **Аналитическое решение:**
   - ω̂ = (XᵀX)⁻¹Xᵀy
   - Получается через взятие производной и приравнивание к нулю

3. **Предсказания:**
   - ŷ = Xω̂
   - Используем найденные коэффициенты для прогнозирования

4. **Преимущества:**
   - Точное решение в один шаг
   - Математически обоснованный метод
   - Не требует итерационной оптимизации
